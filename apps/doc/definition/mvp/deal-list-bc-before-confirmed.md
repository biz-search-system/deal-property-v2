# 案件管理画面（BC確定前案件一覧）要件定義書

## バージョン情報

- **作成日**: 2025-10-25
- **バージョン**: 1.0
- **対象フェーズ**: MVP（最小限の機能を実装した初期バージョン）

---

## 1. 画面概要

### 1.1 画面の目的

BC確定前の案件を一覧表示し、進捗を管理する画面です。営業担当者と事務担当者が日常的に使用し、案件の状況を把握・更新します。

### 1.2 想定利用者

- 営業担当者
- 事務担当者
- 経営者

### 1.3 画面のURL

`/properties/unconfirmed`

---

### 2.2 画面要素の詳細

#### 2.2.1 レイアウト全体構成

**SidebarProvider による3層構造**:

- **AppSidebar**: 左側の固定ナビゲーションメニュー
- **SiteHeader**: 上部の共通ヘッダー（高さ: 48px）
- **SidebarInset**: メインコンテンツエリア

#### 2.2.2 ヘッダー（SiteHeader）

**構成要素**:

- **左側エリア**:
  - サイドバートグル（≡ アイコン）
  - セパレーター（|）
  - パンくずリスト（例: 案件管理 / 業者確定前）

- **右側エリア**:
  - 新規案件登録ボタン（Plus アイコン付き）
  - ※ユーザー情報等はサイドバー内に配置

**技術仕様**:

- 高さ: `var(--header-height)` = 48px
- 背景: `bg-background`
- 下部ボーダー: `border-b`
- スティッキー配置

#### 2.2.3 新規案件登録ボタン

- **配置**: SiteHeader 右端（`ml-auto`）
- **表示条件**: `pathname === "/properties/unconfirmed"` の時のみ表示
- **遷移先**: `/properties/new`
- **デザイン**: Button コンポーネント（Primary variant）
- **アイコン**: Plus（lucide-react）
- **テキスト**: 「新規案件登録」
- **実装**: Link コンポーネントでラップ（Next.js）

#### 2.2.4 パンくずリスト

- **実装**: Breadcrumb コンポーネント（shadcn/ui）
- **表示例**: 「案件管理 / 業者確定前」
- **パス変換ルール**:
  - `/properties` → 「案件管理」
  - `/unconfirmed` → 「業者確定前」
- **リンク可否**: `/properties` は非リンク（ページが存在しない）

#### 2.2.5 集計エリア

**レイアウト**:

- 2カラムのグリッドレイアウト（`grid-cols-2`）
- 各カラムは均等幅
- カード風のデザイン（`bg-muted/50`、`border`、`rounded-lg`）

**表示項目**:

| 項目           | 説明                   | 表示位置 | ラベルスタイル                  | 値スタイル          |
| -------------- | ---------------------- | -------- | ------------------------------- | ------------------- |
| 利益見込み合計 | BC確定前案件の利益合計 | 左カラム | `text-xs text-muted-foreground` | `text-sm font-bold` |
| 件数           | BC確定前案件の総件数   | 右カラム | `text-xs text-muted-foreground` | `text-sm font-bold` |

**実装仕様**:

- パディング: `p-3`
- ギャップ: `gap-4`
- 値の計算: `useMemo` で最適化
- 通貨フォーマット: 万円単位で表示（例: 1,234万）

**将来の拡張（MVP後）**:

- 仲介会社ごとの件数表示
- 総件数と内訳の同時表示
- エクスポート機能（CSV ダウンロード）
- 検索・フィルタ機能

### 3.2 表示仕様詳細

#### 3.2.1 固定列

スクロール時に以下の列を固定表示：

- 担当（左端）
- 物件名（左側）
- 号室（左側）
- 操作（右端）

#### 3.2.2 文字数制限と省略表示

| 項目     | 最大表示文字数 | 省略時の処理           |
| -------- | -------------- | ---------------------- |
| 契約形態 | 5文字          | ツールチップで全文表示 |
| B会社    | 5文字          | ツールチップで全文表示 |
| 仲介     | 5文字          | ツールチップで全文表示 |
| 進捗     | 6文字          | -                      |
| 備考     | 20文字         | ツールチップで全文表示 |

#### 3.2.3 インライン編集可能項目

##### 進捗（progress_status）

**編集方法**:

1. セルをクリック
2. プルダウンメニューが表示
3. 選択肢から選択
4. 自動保存

**選択肢**:

- BC確定前
- 契約CB待ち
- BC契約待ち
- 決済日待ち
- 精算CB待ち
- 決済待ち
- 決済完了

**注意事項**:

- 「BC確定前」以外を選択すると、この画面から自動的に除外される
- 除外前に確認ダイアログを表示

##### 書類（document_status）

**編集方法**:

1. セルをクリック
2. プルダウンメニューが表示
3. 選択肢から選択
4. 自動保存

**選択肢**:

- 営業依頼待ち
- 書類取得中
- 全書類取得完了

##### 備考（notes）

**編集方法**:

1. セルをクリック
2. テキストエリアが表示
3. 編集後、Enterキーまたはフォーカスアウトで保存
4. Escキーでキャンセル

### 3.3 操作メニュー（三点アイコン）

クリック時に以下のメニューを表示：

| メニュー項目 | 機能                   | アイコン |
| ------------ | ---------------------- | -------- |
| 詳細         | 案件詳細画面へ遷移     | 👁       |
| 編集         | 案件編集モーダルを表示 | ✏️       |
| 削除         | 削除確認後、案件を削除 | 🗑       |

## 6. ソート機能

### 6.1 ソート可能項目

- 担当
- オーナー
- B会社
- 仲介会社
- 進捗
- 書類

### 6.2 デフォルトソート

- 更新日時の降順（最新が上）

### 6.3 ソート操作

- カラムヘッダーをクリックで昇順/降順切り替え
- ソート中の項目に▲▼アイコンを表示

## 7. ページネーション

### 7.1 表示件数

- デフォルト: 50件
- 選択可能: 20件 / 50件 / 100件 / 200件

### 7.2 ページ移動

- 前へ / 次へボタン
- ページ番号の直接入力
- 最初 / 最後へのジャンプ

## 8. データ更新

### 8.1 自動更新

- 30秒ごとに自動リフレッシュ（設定で変更可能）
- 他ユーザーの変更をリアルタイム反映

### 8.2 手動更新

- 更新ボタンで即座にリフレッシュ

---

## 9. エクスポート機能

### 9.1 エクスポート形式

- CSV形式
- 文字コード: UTF-8 BOM付き（Excel対応）

### 9.2 エクスポート内容

- 現在のフィルタ条件を適用した結果
- 全項目を含む（表示/非表示に関わらず）

## 11. バリデーション

### 11.1 進捗変更時

- 「BC確定前」から他のステータスへ変更時に確認ダイアログ
- 「この案件は業者確定後画面に移動します。よろしいですか？」

### 11.2 削除時

- 確認ダイアログ表示
- 「この案件を削除してもよろしいですか？この操作は取り消せません。」

---

## 12. エラー処理

### 12.1 通信エラー

- エラーメッセージをトースト表示
- 再試行ボタンを表示

### 12.2 権限エラー

- 「この操作を実行する権限がありません」と表示

### 12.3 同時編集の競合

- 楽観的ロックを実装
- 「他のユーザーが編集中です。画面を更新してください」

- 1ページあたり最大200件
