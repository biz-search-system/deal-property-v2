# クライアント売上管理ルール

> このドキュメントは、売上計上ルールとホワイトボード（スプレッドシート）の定義を説明します。

## 1. 売上計上ルール

### 1.1 基本ルール

AB間契約締結後、C業者が確定するまでに時間がかかるため、以下のルールで売上計上月を決定する。

**ルール**: 当月契約した物件を**翌月末日までに**C業者を確定できれば、**当月の売上**として計上

### 1.2 計上月の判定例

| AB契約日 | C業者確定日 | 売上計上月 | 説明 |
|----------|-------------|------------|------|
| 8月15日 | 8月30日 | 8月 | 翌月末（9月末）以前に確定 |
| 8月15日 | 9月27日 | 8月 | 翌月末（9月末）以前に確定 |
| 8月15日 | 10月3日 | 9月 | 翌月末を超過、確定月の前月 |
| 8月15日 | 11月30日 | 10月 | 翌月末を超過、確定月の前月 |

### 1.3 判定ロジック

```
IF C業者確定日 <= AB契約月の翌月末日 THEN
  売上計上月 = AB契約月
ELSE
  売上計上月 = C業者確定月 - 1ヶ月
```

## 2. ホワイトボード（スプレッドシート）定義

### 2.1 構成

各月のシートに以下のセクションがある:

1. **今月仕入物件一覧**
   - 当月契約で業者確定済み案件
   - 各営業の売上数字を反映

2. **未確定案件一覧**
   - 前月以前契約で業者未確定の案件

### 2.2 ステータス定義

| ステータス | 説明 |
|------------|------|
| 未確定 | AB契約済、C業者未確定（出口管理表の▲or😭状態） |
| 確定 | C業者確定済（出口管理表の〇状態） |
| 次月 | 翌月末までに確定せず、次月シートへ移動 |
| 違約入金済 | 売主から違約金入金完了。着金月に売上計上 |
| 違約予定 | 違約方向で確定、違約金入金待ち（未確定と同様の扱い） |
| 弁護士 | 内容証明受領または起訴中。方向性未定なら翌月以降未確定へ |

## 3. 通常パターンの運用フロー

### 3.1 当月契約→翌月末までに確定（通常パターン）

```
1. AB契約締結
2. 当月の「今月仕入物件一覧」に入力
   - 担当/物件名/仕入れ金額等
   - プルダウン: 未確定
3. 業者確定
4. プルダウン: 確定 に変更
5. 各営業の数字に売上反映
```

### 3.2 当月契約→翌月末までに未確定（遅延パターン）

```
例: 6月15日契約、7月末時点で未確定

1. 6月シート「今月仕入物件一覧」に入力
2. 7月末時点で未確定
3. ステータス: 次月 に変更
4. 7月シート「未確定案件一覧」にコピペ移動
5. 8月中に業者確定した場合
   → 売上計上月は7月（前月）
   → 7月シート「未確定案件一覧」から「今月仕入物件一覧」へ移動
   → 当月仕入物件と区別するため、下から順に追加
```

## 4. イレギュラーパターン

### 4.1 違約入金済

- **条件**: 売主から違約金が入金された
- **売上計上**: 着金確認月
- **運用**:
  - 当月に違約方向+当月入金 → 当月売上
  - 当月に違約方向+翌月入金 → 翌月売上

### 4.2 違約予定

- **条件**: 違約方向が確定、入金待ち
- **扱い**: 未確定と同様
- **運用**: 当月入金なければ翌月「未確定案件一覧」へ移動

### 4.3 弁護士案件

- **条件**: 売主から内容証明受領、またはこちらから起訴中
- **扱い**: 業者確定不可
- **運用**: 当月に方向性未定なら翌月「未確定案件一覧」へ移動

## 5. ステータス遷移図

```
[AB契約締結]
    ↓
[未確定]
    ├─→ [確定] → 当月売上計上
    ├─→ [次月] → 翌月シートへ移動
    │         └─→ [確定] → 前月売上計上
    │         └─→ [次月] → さらに翌月へ...
    ├─→ [違約予定]
    │         └─→ [違約入金済] → 着金月売上計上
    │         └─→ [次月] → 翌月シートへ移動
    └─→ [弁護士]
              └─→ [確定] → 確定月前月売上計上
              └─→ [次月] → 翌月シートへ移動
```
