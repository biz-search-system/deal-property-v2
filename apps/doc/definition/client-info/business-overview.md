# クライアント業務概要書

> このドキュメントは、Agentがシステム開発時に参照するためのクライアント業務の詳細説明書です。

## 1. 会社概要

### 1.1 事業内容

- **主要業務**: 1R（ワンルーム）投資用マンションの仲介
- **営業形態**: 物上げ（ぶつあげ）営業
- **営業スタイル**: テレアポ（電話営業）
- **収益モデル**: 相場以下での売却仲介による差益

### 1.2 物上げとは

既に1Rを所有しているオーナー（売主）に対してアプローチを行い、売却を促す営業手法。

## 2. 取引構造

### 2.1 取引関係者の定義

| 記号 | 役割 | 説明 |
|------|------|------|
| A | 所有者（売主） | 物件を所有しているオーナー |
| B | グループ会社 | クライアント企業のグループ会社（買主役） |
| C | 買取再販業者 | Bから物件を購入する業者 |
| D | エンド | 一般買主（最終購入者） |
| 仲介 | クライアント企業 | レイジット（仲介業務を担当） |

### 2.2 取引フロー

```
[A: 所有者]
    ↓ 仲介（レイジット）がテレアポでアプローチ
    ↓ 相場以下の価格で売却を促す
[AB間売買契約]
    ↓ Bはグループ会社だが、Aには買取とは説明しない
    ↓ Aから仲介手数料を取得
[マイソク配布]
    ↓ 物件資料を各買取業者(C)に送付
    ↓ 買取金額を確認
[BC間売買契約]
    ↓ 金額が問題なければ契約締結
[決済完了]
    ↓ Cが買取後、Dに販売（クライアントの業務範囲外）
```

### 2.3 収益構造の例

```
AB間: 売買代金 1,000万円 + 仲介手数料 39.6万円
BC間: 売買代金 1,200万円（仲介手数料込み）
CD間: 売買代金 1,500万円（参考：Cの利益、管理対象外）

クライアントの利益 = 1,200万 - 1,000万 + 39.6万 = 239.6万円
```

## 3. 業務フロー詳細

### 3.1 全体フロー

```
①マイソク作成・業者配布
    ↓（①と②は前後する場合あり）
②AB間売買契約 締結
    ↓
③物件調査
  ├─ 建物管理から書類取得（重調・管理規約・長計）
  ├─ 賃貸管理から書類取得（賃契・管理委託契約書）
  └─ 役所から書類取得（公課証明・建築概要）
    ↓
④BC契約書・重説作成
    ↓
～決済日確定後～
    ↓
⑤精算書作成
    ↓
⑥司法書士依頼
    ↓
⑦賃貸管理解約
    ↓
⑧決済完了
```

## 4. 用語集

| 重調 | じゅうちょう | 重要事項調査報告書 |

## 5. 現行システム・ツール

### 5.1 使用中のツール

| ツール | 用途 |
|--------|------|
| スプレッドシート | 出口管理表、決済確定管理表、司法書士共有シート |
| Evernote | 調査関係、BC関係、精算書の進捗管理 |
| Excel | マイソク作成 |
| メール | 業者への一括配布、金額回答受付 |
| チャットワーク | 営業から事務への依頼連絡 |

### 5.2 主要スプレッドシート

1. **【レイジット】出口管理表**
   - 物件情報と業者からの回答金額を管理
   - 「出口概要」タブで進捗追跡

2. **決済確定管理表**
   - BC未確定案件の管理
   - 決済確定後の案件追跡

3. **司法書士共有シート**
   - 林先生×レイジット
   - 谷口先生×レイジット
   - 石橋先生×レイジット
   - 買書士 一本化
