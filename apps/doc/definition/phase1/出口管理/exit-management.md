# 出口管理機能 要件定義書

## 1. 機能概要

### 1.1 目的

- マイソク（物件資料）を買取業者に配布し、金額回答を管理する機能。
- 現在Excel「出口管理表」とメールで行っている業務をシステム化し、業者選定の効率化と情報の一元管理を実現する。

### 1.2 システム導入による効果

- マイソク作成・配布の効率化
- 金額回答の自動反映と自動順位付け
- 最適な業者の迅速な選定
- 追いメールによるフォロー漏れ防止
- 業者分析によるデータドリブンな意思決定
- 複数人での同時アクセス・編集
- 変更履歴の自動記録

## 6. 回答フォーム（業者向け）

### 6.1 概要

業者がマイソクを確認し、買取金額を回答するフォームです。認証不要でアクセス可能。

### 6.2 物件情報表示

マイソクの内容を表示（物件名、所在地、築年月、専有面積、構造、階数、現況、家賃、管積、想定利回りなど）。

### 6.3 回答入力項目

| 項目       | 必須 | 説明                 |
| ---------- | ---- | -------------------- |
| 買取金額   | ○    | 買取希望金額（万円） |
| 想定利回り | -    | 自動計算または手入力 |
| 回答者名   | ○    | 担当者名             |
| 連絡先     | -    | 電話番号またはメール |
| 条件・備考 | -    | 金額の条件や備考     |
| 有効期限   | -    | 金額の有効期限       |

### 6.4 回答後の動作

- 回答完了メッセージを表示
- 回答内容の確認画面
- 金額変更リンク（期限内であれば再回答可）

### 6.5 セキュリティ

- 業者ごとにユニークなトークンを発行
- トークンには有効期限を設定
- 1つのトークンで複数回の回答更新が可能

### 6.6 画面URL

| 画面         | URL          |
| ------------ | ------------ |
| 回答フォーム | `/r/{token}` |

## 7. メール配布

### 7.1 概要

マイソクを業者にメール配布する機能です。追いメール（リマインダー）にも対応。

### 7.2 配布メール内容

- 物件情報の概要
- マイソクPDF添付
- 回答フォームへのリンク
- 回答期限
- 担当者連絡先

### 7.3 配布方法

1. 配布対象の業者を選択（チェックボックス/グループ選択）
2. 配布メッセージを入力（テンプレート利用可）
3. 一括送信を実行
4. 各業者に固有のトークン付きURLを発行

### 7.4 追いメール

- 未回答業者を自動抽出
- リマインダーメッセージを送信
- 送信回数を記録

### 7.5 配布履歴

| 記録項目       | 説明                   |
| -------------- | ---------------------- |
| 配布日時       | 送信した日時           |
| 配布先業者     | 送信した業者一覧       |
| 配布者         | 送信を実行したユーザー |
| 配布メッセージ | 送信したメッセージ内容 |
| 追いメール回数 | リマインダー送信回数   |

### 7.6 画面URL

| 画面     | URL                          |
| -------- | ---------------------------- |
| 業者配布 | `/exits/{exitId}/distribute` |

## 9. 案件連携

### 9.1 概要

出口管理と案件管理（MVP）を連携させる機能です。

### 9.2 連携パターン

**パターン1: 案件登録時に出口管理を自動作成**

1. 案件を新規登録
2. 出口管理レコードが自動作成され、案件と紐付け
3. 案件情報（物件名、号室、担当者等）を出口管理に引き継ぎ

**パターン2: 案件詳細から出口管理を開始**

1. 案件詳細画面から「出口管理を開始」ボタン
2. 案件情報を引き継いでマイソク作成画面へ
3. 出口管理レコードと案件を紐付け

**パターン3: 出口管理から案件を作成**

1. 出口管理で業者確定後
2. 「案件を作成」ボタンで案件作成画面へ
3. 出口情報を引き継いで案件作成

### 9.3 自動更新項目

業者確定時に案件に反映される項目：

| 出口管理 | 案件     |
| -------- | -------- |
| 確定業者 | 買取業者 |
| 確定金額 | 出口金額 |

### 9.4 案件画面からの出口情報表示

- 案件詳細画面に出口管理へのリンク
- 回答状況のサマリー表示（アッパー、2番手、3番手）

### 9.5 案件詳細で管理する追加項目

出口管理表の「出口概要シート」「買仲案件シート」で管理していた項目は、案件詳細で管理します。

#### 出口概要シート由来の項目

| 項目         | フィールド名               | 説明                     | Excel管理元    |
| ------------ | -------------------------- | ------------------------ | -------------- |
| ABサイト     | ab_settlement_date         | AB間の決済期日           | 出口概要シート |
| BCサイト希望 | bc_settlement_date_desired | BC間の決済希望日         | 出口概要シート |
| 借入状況     | loan_status                | ローン状況（借入待ち等） | 出口概要シート |

※A契約日、備考は既にMVP案件管理で対応済み

#### 買仲案件シート由来の項目

| 項目       | フィールド名       | 説明             | Excel管理元    |
| ---------- | ------------------ | ---------------- | -------------- |
| 管理形態   | management_type    | 管理会社形態     | 買仲案件シート |
| 仕入れ業者 | purchase_broker    | 仕入れ元業者     | 買仲案件シート |

※契約形態、仲介会社、配布業者は既にMVP案件管理または出口管理で対応済み

## 10. 権限

| ロール       | コード | 説明                                       |
| ------------ | ------ | ------------------------------------------ |
| 経営         | owner  | 組織オーナー。全機能へのフルアクセス権限   |
| 営業管理者   | admin  | 営業部門の管理者。営業業務の管理権限を持つ |

| 事務管理者   | admin  | 事務部門の管理者。事務業務の管理権限を持つ |
| 事務メンバー | member | 事務部門の一般メンバー。サポート業務を担当 |

| ------------------ | -------------- | --------------------------------------------- |

| 出口管理編集全件   | 営業（member） | 自身が担当でない案件は閲覧のみ、編集は要承認  |
| ステータス変更     | 事務（member） | 「確定」以外のステータスのみ変更可能          |
| マイソク編集全件   | 営業（member） | 担当外は上長承認後のみ編集可能                |
| マイソク編集全件   | 事務（member） | 担当外は上長承認後のみ編集可能                |
| メール一括配布     | 事務（member） | 10件以上の一括配布は上長確認が必要            |
| 業者確定操作       | 営業（member） | 500万円以下の案件のみ。超過時は上長承認必要   |
| 業者確定操作       | 事務（admin）  | 緊急時のみ。通常は営業部門が実施              |
| 業者新規登録       | 営業（member） | 登録後、adminによる確認・有効化が必要         |
| 業者情報編集       | 営業（member） | メールアドレス・電話番号等の軽微な修正のみ    |
| 期間カスタム設定   | 事務（member） | プリセット期間（3ヶ月/6ヶ月/1年）のみ選択可能 |
| データエクスポート | 営業（member） | 月次レポートのみ。任意期間は要上長承認        |

### 10.5 データアクセス範囲

| ロール         | 閲覧可能データ範囲                             |
| -------------- | ---------------------------------------------- |
| 経営（owner）  | 組織全体のすべてのデータ                       |
| 営業（admin）  | 営業チームおよび関連する全データ               |
| 営業（member） | 自身が担当する案件 + 営業チームの共有データ    |
| 事務（admin）  | 事務チームおよび関連する全データ               |
| 事務（member） | 事務処理に必要な範囲のデータ（個人情報は制限） |

### 10.6 特殊操作の承認フロー

#### 業者確定（高額案件）

```text
営業（member）が確定申請 → 営業（admin）が承認 → 確定処理実行
```

#### 業者マスタ一括インポート

```text
事務（admin）がインポート実行 → システムが重複チェック → 経営（owner）が最終承認
```

#### 出口管理削除

```text
営業（admin）が削除申請 → 経営（owner）が承認 → 論理削除実行
```

### 10.7 監査ログ

以下の操作は監査ログに記録されます。

| 操作カテゴリ | 記録対象                           |
| ------------ | ---------------------------------- |
| 認証         | ログイン/ログアウト                |
| データ変更   | 作成/更新/削除すべて               |
| 権限操作     | ロール変更、権限付与/剥奪          |
| 重要操作     | 業者確定、ステータス変更、一括配布 |
| エクスポート | CSV/PDFエクスポート                |
| 承認フロー   | 申請/承認/却下                     |
